## 📖 О проекте (About the Project)
**Multi-functional Web Calculator** — это современное и многофункциональное веб-приложение, объединяющее в себе четыре различных типа калькуляторов.

Изначально проект был разработан в рамках университетской курсовой работы, но в последствии прошел этап глубокого код-ревью, рефакторинга и оптимизации. Приложение демонстрирует чистый интерфейс, работу с математическими библиотеками, парсинг регулярных выражений, работу со строками и динамическую отрисовку графиков на Canvas.

## ✨ Ключевой функционал (Features)
Приложение состоит из 4 независимых модулей (вкладок):

*   🧮 **Обычный (Инженерный) калькулятор**
    *   Поддерживает базовые арифметические операции и расширенные математические функции (тригонометрия, логарифмы, факториалы).
    *   Безопасное вычисление выражений с помощью `math.js`, защита от деления на ноль и некорректного синтаксиса.
    *   Ведет историю вычислений.

*   🔢 **Калькулятор систем счисления (Numeral Systems)**
    *   Конвертирует числа между любыми системами счисления от двоичной (base-2) до тридцатишестиричной (base-36).
    *   Показывает подробные математические шаги перевода (умножение на степени и деление с остатком).
    *   Включает строгую валидацию на недопустимые символы в рамках выбранной базы.

*   💱 **Конвертер валют (Currency Converter)**
    *   Позволяет переводить средства между популярными мировыми валютами (включая USD, EUR, RUB, CNY и др.).
    *   Встроенный график колебания курсов валют (на базе `Chart.js`) с возможностью выбора исторического периода (Месяц, Год, Всё время).
    *   Динамический расчет кросс-курсов валют.

*   📈 **Построитель графиков (Graphing Calculator)**
    *   Отрисовывает 2D-графики как явных (например, `y = sin(x) + x^2`), так и неявных функций (например, `x^2 + y^2 = 1`).
    *   Усовершенствованный парсинг тригонометрических и логарифмических выражений с помощью RegEx.
    *   Интерактивный Canvas: поддерживает масштабирование скроллом, перетаскивание (Drag & Drop) координатной плоскости и кастомные цвета линий графиков.

## 🛠 Стек технологий (Built With)
Проект написан на чистом (Vanilla) стеке веб-технологий без использования тяжелых фреймворков:
*   **HTML5** / **CSS3** — структура и кастомная стилизация, поддержка светлой и темной тем.
*   **Vanilla JavaScript (ES6+)** — логика вычислений и DOM-манипуляций.
*   [Math.js](https://mathjs.org/) — мощная библиотека для оценки математических выражений.
*   [Chart.js](https://www.chartjs.org/) — для отрисовки графиков истории курсов в конвертере валют.
*   [Plotly.js](https://plotly.com/javascript/) — подключен для потенциального расширения возможностей визуализации данных.

## 💻 Как запустить локально (Getting Started)
Проект не требует сложной сборки (Node.js, Webpack и т.д.). Вы можете запустить его в два клика.

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/ВАШ_НИК/НАЗВАНИЕ_РЕПОЗИТОРИЯ.git
   ```
2. **Откройте проект:**
   * Просто откройте файл `index.html` в любом современном браузере.
   * *Или (рекомендуется):* Откройте папку проекта в VS Code и используйте расширение **Live Server**, чтобы запустить локальный сервер для корректной работы всех путей к картинкам и скриптам.
